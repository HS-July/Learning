# Calculator

Python으로 만든 계산기 프로그램입니다.

## 기능

- 더하기 (+)
- 빼기 (-)
- 곱하기 (*)
- 나누기 (/)
- 제곱 (**)
- 나머지 (%)
- 복리 계산

## 실행 방법

```bash
python calculator.py
```

## 사용 예시

```
===== 계산기 =====
1. 더하기 (+)
2. 빼기 (-)
3. 곱하기 (*)
4. 나누기 (/)
5. 제곱 (**)
6. 나머지 (%)
7. 복리 계산
0. 종료

연산을 선택하세요: 7

원금: 10000000
연이율 (%): 5
기간 (년): 10

--- 복리 계산 결과 ---
원금:       10,000,000원
연이율:     5%
기간:       10년
최종 금액:  16,288,946원
수익:       6,288,946원
```
